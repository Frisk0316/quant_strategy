# Research Search Log

Search date: 2026-04-27, Asia/Taipei.

The first pass used broad discovery searches across arXiv, SSRN, publisher
pages, journal landing pages, and general web search. The database favors papers
that either define a reusable trading signal, explain a structural edge, or
warn against a common false edge.
