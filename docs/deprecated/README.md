# Deprecated Documents

Deprecated documents are obsolete or replaced, but retained because existing discussions, PRs, or ADRs may still reference them.

Deprecated documents must not be used as implementation authority.

Before moving a document here:

- Set `status: deprecated` in lifecycle metadata when practical.
- Set `superseded_by` to the replacement path or ADR id.
- Add a visible note near the top of the document naming the replacement.
- Ensure the replacement document is `current` or `accepted`.
